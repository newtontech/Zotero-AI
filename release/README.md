# Zotero AI Newton 0.0.1-beta

Prebuilt XPI and update manifest for users who prefer to install without building the project.

## Files

- `zotero-ai-newton-0.0.1-beta.xpi`: ready-to-install package.
- `update-beta.json`: update feed pointing to the raw GitHub download.

## Install steps

1. Download `zotero-ai-newton-0.0.1-beta.xpi` (or use the raw link: https://raw.githubusercontent.com/newtontech/Zotero-AI-Newton/main/release/zotero-ai-newton-0.0.1-beta.xpi).
2. In Zotero, open **Tools → Add-ons**, click **Install Add-on From File…**, and select the downloaded XPI.
3. Open **Edit → Preferences → Zotero AI Newton** and paste your OpenAI or DeepSeek API key and model name.
4. Restart Zotero if prompted. Future beta updates will be picked up automatically from `release/update-beta.json`.
